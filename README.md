# Pelican Panel Plugins

A collection of plugins for [Pelican Panel](https://github.com/pelican-dev/panel).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [server-documentation](./server-documentation) | Attach markdown documentation to servers with versioning and role-based access |

## Installation

Each plugin has its own installation instructions in its README. Generally, plugins can be installed via the Pelican Panel admin interface or manually placed in the `plugins/` directory.
